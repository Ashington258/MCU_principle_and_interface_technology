# 数制和编码

## 1 数制

1. 二进制:$1001B = 1\times2^3 + 0\times2^2 + 0\times2^1 + 1\times2^0 = 9D$
2. 十进制:$1135D = 1\times10^3 + 1\times10^2 + 3\times10^1 + 5\times10^0$
3. 十六进制:$1C5H = 1\times16^2 + 14\times16^1 + 5\times16^0 = 453D$
